Esta pasta é destinada ao armazenamento das imagens dos produtos (pizzas).

Para adicionar uma nova imagem de pizza:
1. Salve o arquivo de imagem nesta pasta
2. Use o caminho relativo: ~/Images/Pizzas/nome_do_arquivo.jpg
3. Ou use o caminho absoluto: /Images/Pizzas/nome_do_arquivo.jpg

Formatos suportados: .jpg, .jpeg, .png

Recomendações:
- Use nomes descritivos para os arquivos (ex: pizza_margherita.jpg)
- Tamanho recomendado: 600x600 pixels 
- Peso recomendado: máximo 500KB por imagem
