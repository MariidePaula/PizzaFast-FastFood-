Esta pasta é destinada ao armazenamento das imagens das bebidas.

Para adicionar uma nova imagem de bebida:
1. Salve o arquivo de imagem nesta pasta
2. Use o caminho relativo: ~/Images/Bebidas/nome_do_arquivo.jpg
3. Ou use o caminho absoluto: /Images/Bebidas/nome_do_arquivo.jpg

Formatos suportados: .jpg, .jpeg, .png

Recomendações:
- Use nomes descritivos para os arquivos (ex: coca_cola_lata.jpg)
- Tamanho recomendado: 300 × 300 px pixels 
- Peso recomendado: máximo 500KB por imagem
