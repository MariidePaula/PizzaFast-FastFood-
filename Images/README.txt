Esta pasta é destinada ao armazenamento das imagens dos produtos.

Estrutura de pastas:
- Pizzas/ - Imagens dos produtos (pizzas)
- Bebidas/ - Imagens das bebidas

Para adicionar uma nova imagem:
1. Salve o arquivo de imagem na pasta apropriada (Pizzas ou Bebidas)
2. Use o caminho relativo: ~/Images/Pizzas/nome_do_arquivo.jpg ou ~/Images/Bebidas/nome_do_arquivo.jpg
3. Ou use o caminho absoluto: /Images/Pizzas/nome_do_arquivo.jpg ou /Images/Bebidas/nome_do_arquivo.jpg

Formatos suportados: .jpg, .jpeg, .png

OBS: Alguns cards não possuem imagens intencionalmente, permitindo que o usuário personalize o conteúdo de acordo com sua preferência. :)
